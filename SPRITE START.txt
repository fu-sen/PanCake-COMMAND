SPRITE START
0.8〜
-----------------------------------------------------------
[ 機能 ]
スプライトモードへの開始・終了を行います。

[ コマンド 書式 ]
PANCAKE SPRITE START <表示>

[ 例 ]
PANCAKE SPRITE START 05
PANCAKE SPRITE START 1A

[ コマンド 書式 ]
80 04 06 <表示>

[ 例 ]
80 04 06 05
80 04 06 1A

[ 解説 ]
<表示> が 00〜07 の場合は PanCake で用意されている組み込み画像になります。
<表示> が 10〜1F の場合は 0〜F の色番号が背景になります。
<表示> を FF とする事でスプライトモードを終了します。

スプライトモードで動作している間は LINE・IMAGE などの線画機能は使用できません。
