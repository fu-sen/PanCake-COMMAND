RESET
0.9～
-----------------------------------------------------------
[ 機能 ]
初期状態に戻します。

[ 共通コマンド 書式 ]
PANCAKE RESET
PC RESET (1.0～)

[ 共通コマンド 例 ]
PANCAKE RESET
PC RESET (1.0～)

[ IchigoCake BASIC コマンド 書式 ]
PC.RESET

[ IchigoCake BASIC コマンド 例 ]
PC.RESET

[ IchigoCake JavaScript コマンド 書式 ]
reset()

[ IchigoCake JavaScript コマンド 例 ]
reset()

[ バイナリコマンド 書式 ]
128 3 13
#80 #03 #0D

[ バイナリコマンド 例 ]
128 3 13
#80 #03 #0D

[ 解説 ]
画面・スプライト・音など、全てを初期状態へ戻します。
